## Installation

```sh
go get -u github.com/joseluisq/go-tspath
```

## Usage

```sh
# after the tsc compiling just run:
go-tspath -config=./tsconfig.json
```

## API

```sh
~> go-tspath --help
Usage of go-tspath:
  -config string
    	Specifies the Typescript configuration file. (default "./tsconfig.json")
```
